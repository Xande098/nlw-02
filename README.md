## :information_source: How to use

To clone and run this application, you'll need [Git](https://git-scm.com) + [Yarn](https://classic.yarnpkg.com/en/docs/install/#windows-stable) installed on your computer.

From your command line:

### Install API
```bash
# Clone this repository
$ git clone https://github.com/Xande098/nlw-02.git

# Go into the repository
$ cd nlw-02/server

# Install dependencies
$ yarn

# Run Migrates
$ yarn knex:migrate

# Run Seeds
$ yarn knex:seed

# Start server
$ yarn dev

# running on port 3333
```

### Install Front-end

```bash
# Clone this repository
$ git clone https://github.com/Xande098/nlw-02.git

# Go into the repository
$ cd nlw-02/web

# Install dependencies
$ yarn

# Run
$ yarn start

# running on port 3000
```

### Install Mobile

```bash
# Clone this repository
$ git clone https://github.com/Xande098/nlw-02.git

# Go into the repository
$ cd nlw-02/mobile

# Install dependencies
$ yarn

# Run
$ yarn start

# Expo will open, just scan the qrcode on terminal or expo page

# If some problem with fonts, execute:
$ expo install expo-font @expo-google-fonts/ubuntu @expo-google-fonts/roboto

```
